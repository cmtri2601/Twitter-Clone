# Twitter Clone Project

## Tech stack

- Env: Node.js
- Lib: Express.js, Typescript
- Database: Mongo

## Return format

- header: status
- body:
  - message: CommonMessage
  - detail: detail of response - specific message
  - errors: error of list error
  - data: response value
